# File-Based Data Management System (Python)

## Project Overview

This project is a menu-driven file management system developed using Python. It allows users to perform basic file operations such as create, read, update, and delete (CRUD) on files, simulating a simple data management system.

## Technologies Used

* Python
* pathlib module
* OS module

## Features

* Create new files with user input
* Read and display file contents
* Update files (rename, overwrite, append)
* Delete files
* Directory traversal and file listing

## Functionalities

* Uses `pathlib` for handling file paths
* Implements CRUD operations on files
* Provides a menu-driven interface for user interaction
* Handles errors using exception handling

## How to Run

1. Make sure Python is installed
2. Run the program:

   ```
   python main.py
   ```

## Files

* main.py → Main program file
